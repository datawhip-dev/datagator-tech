Get Started in 3 Steps

All you need is a bind-mount and Docker or Podman.

  -v /your/data:/mnt:ro \
  -v ~/.DataGator.id:/etc/DataGator.id:ro \
  datagator scan /mnt

Want to detect changes over time?

./drift_detector.sh
